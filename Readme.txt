# BlueSoftTest

# Este projeto realiza testes na interface do site Web https://magento.softwaretestingboard.com/
# Linguagem utilizada: Cypress
# Para executar o projeto:
# 1 - Instale a versão mais recente do Cypress (npm install cypress --save-dev)
# 2 - Clone o projeto através do link: https://github.com/ThandeSperatti/blueSoftTeste
# 3 - Inicie o projeto utilizando o comando npx cypress open
# 4 - Selecione a opção E2E Testing
# 5 - Selecione o navegador desejado
# 6 - Clique na opção "Start E2E Testing in <navegador>"
# 7 - Será aberta uma nova janela com o arquivo onde constam os testes "tests.cy.js"
